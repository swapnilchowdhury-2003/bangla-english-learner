# bangla-english-learner
Daily Bangla → English translation practice app with grammar explanations in Bangla
A single-file offline-ready web app for Bengali speakers learning English grammar through daily translation exercises.

## Features
- Translate Bangla sentences into English across 4 grammar modules: Present, Past, Future, and Have/Has/Had
- 30-second thinking timer that stops the moment you start typing
- Wrong answers trigger a full Bangla explanation — tense name, why it's used, subject/verb breakdown, and color-coded syntax structure
- Streak counter saved in localStorage
- Runs as a standalone HTML file — no install, no internet needed, no dependencies

## Grammar Covered
- Simple Present / Present Continuous / Present Perfect
- Simple Past / Past Continuous / Used to
- Simple Future / Future Continuous / Be going to
- Have / Has / Had — possession and perfect tenses
- Question forms for all tenses

## How to Use
Download index.html and open it in any browser. That's it.

## Tech
Pure HTML + CSS + JavaScript. No frameworks. No build step. Single file.
